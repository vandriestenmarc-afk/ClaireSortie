Une IA qui montre ses preuves

      
Trois garde-fous avant toute utilité

      

        
1
Extraire, pas inventer
Les informations absentes restent absentes. Elles deviennent des points à faire confirmer.

        
2
Relier chaque résultat
Chaque élément affiche le passage exact du document qui le justifie.

        
3
Vérifier la compréhension
Le quiz reprend les sources pour aider l’utilisateur à repérer ce qu’il doit revoir.

      

      
Deux moteurs, une même règle de sécurité : Render exécute GPT‑5.6 côté serveur avec la clé protégée. GitHub Pages reste le secours local. Dans les deux cas, les informations absentes ne sont pas inventées.
