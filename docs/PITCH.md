# ClairSortie — concise pitch

After discharge, patients may receive medication changes, follow-up appointments, home-care instructions, and warning signs at the same time. ClairSortie turns a synthetic discharge document into a readable action plan while keeping every item connected to its exact source passage.

Unlike a generic summary, ClairSortie treats uncertainty as a first-class output: missing details are surfaced for confirmation rather than invented. A short source-linked quiz helps users check what they understood.

The public demo is private-by-design and runs locally with no credentials. The repository also includes a meaningful GPT-5.6 server path using strict structured output, `store: false`, prompt-injection resistance, and fail-closed exact-quote validation.

Codex built the modular product foundation, evidence-preserving extraction, accessible interface, quiz, exports, tests, and GPT integration direction. The result is a transparent prototype for safer hospital-to-home communication—not a medical device or treatment tool.
